# LAIA Tech — Landing Page
Arquitetura simples (HTML/CSS/JS) com animações de rolagem, parallax e fontes modernas.

## Como usar
1. Publique esta pasta (ou o ZIP) no Netlify/Vercel/GitHub Pages.
2. Troque `assets/brain.svg` pelo seu cérebro em PNG/SVG se desejar.
3. Ajuste cores em `styles.css` (variáveis CSS no topo).
4. Edite textos/links em `index.html`.
